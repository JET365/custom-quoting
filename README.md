# JET365 Quote Generator

Internal tool for generating branded charter quotes from Avinode data.

## Deploy to Vercel

1. Push this folder to a **private** GitHub repo
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"**
4. Import your repo
5. Vercel auto-detects Vite — just click **Deploy**
6. Done. You'll get a URL like `jet365-quote-tool.vercel.app`

### Optional: Custom domain

In your Vercel project dashboard, go to **Settings > Domains** and add a subdomain like `quote.jet365.com`.

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`
